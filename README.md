# Fitness1

Kısa açıklama: Bu repo fitness uygulaması için başlangıç dosyalarını içerir.

Kurulum:
1. Depoyu klonla:
   git clone https://github.com/piyamip/Fitness1.git
2. İlgili dizine gir:
   cd Fitness1

Hızlı Başlangıç:
- index.html dosyası GitHub Pages için eklendi. Siteyi açmak için Settings → Pages → Branch: main, Folder: / (root) seçin.

İçerik:
- README.md — proje açıklaması
- index.html — basit ana sayfa
